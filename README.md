## 1. 프로젝트 개요

- **프로젝트명**: Windows 타이머 알림 프로그램
- **개발 기간**: 2026.05.31
- **개발환경/기술스택**:
    - Python
    - Tkinter GUI
    - Pystray
    - Winotify
    - PyWin32
    - JSON 데이터 저장

---

## 2. 주요 기능

### (1) 타이머 설정 및 알림

- 사용자 지정 시간 입력
- 기본값 30시간 제공
- 테스트용 초(Seconds) 입력 지원
- 설정 시간 도달 시 Windows 알림 출력
- 알림 완료 시 팝업 메시지 표시

예시:

```
시간(Hours): 30
초(Seconds): 0
```

---

### (2) 실시간 남은 시간 표시

- 현재 남은 시간 실시간 계산
- 1초 단위 자동 갱신
- GUI 화면 내 실시간 표시

예시:

```
남은 시간 : 29시간 58분 12초
```

---

### (3) 시스템 트레이(숨겨진 아이콘) 지원

- 프로그램 종료(X) 시 완전 종료되지 않음
- 시스템 트레이로 최소화
- 트레이 아이콘을 통한 프로그램 제어

기능:

```
창 열기
숨기기
끄기
```

---

### (4) 트레이 아이콘 남은 시간 표시

- 트레이 아이콘 마우스 오버 시 남은 시간 확인 가능
- 1초 단위 자동 갱신

예시:

```
남은 시간: 12시간 35분 21초
```

---

### (5) 자동 저장 및 복구

- 타이머 설정 시 목표 시간 자동 저장
- JSON 파일 기반 데이터 유지
- 프로그램 재실행 시 자동 복구

예시:

```
{
  "target_time":"2026-06-02T15:30:00"
}
```

---

### (6) Windows 시작 시 자동 실행

- 최초 실행 시 시작프로그램 자동 등록
- Windows 로그인 시 자동 실행
- 백그라운드 상태 유지

구현 방식:

```
Startup Folder
↓
PythonTimer.lnk 생성
↓
Windows 부팅 시 자동 실행
```

---

## 3. 차별화된 기술 포인트

- **Windows 시스템 트레이 연동**
    
    → 일반 GUI 프로그램이 아닌 백그라운드 상주형 응용프로그램 구현
    
- **Windows 알림(Notification) 시스템 활용**
    
    → 운영체제 기본 알림 기능과 연동
    
- **실시간 남은 시간 계산**
    
    → 목표 시간 기준으로 실시간 카운트다운 처리
    
- **자동 저장 및 복구 기능**
    
    → 프로그램 종료 및 재부팅 이후에도 타이머 상태 유지
    
- **시작프로그램 자동 등록**
    
    → 사용자의 별도 설정 없이 Windows 시작 시 자동 실행
    
- **GUI + 백그라운드 서비스 구조**
    
    → 화면 표시와 백그라운드 작업을 동시에 처리하는 구조 구현

## 4. 구현 내용

### (1) 타이머 로직 구현

- 현재 시간과 목표 시간 계산
- timedelta 기반 남은 시간 산출
- 실시간 카운트다운 처리

예시:

```
target_time=datetime.now()+timedelta(hours=30)

remain=target_time-datetime.now()
```

---

### (2) Windows 알림 구현

- Winotify 라이브러리 활용
- 타이머 완료 시 운영체제 알림 생성

예시:

```
Notification(
app_id="Python Timer",
title="타이머 완료",
msg="설정한 시간이 지났습니다!"
)
```

---

### (3) 시스템 트레이 구현

- Pystray 라이브러리 활용
- 아이콘 생성 및 메뉴 제공
- GUI 최소화 상태에서도 동작 유지

예시:

```
창 열기
숨기기
끄기
```

---

### (4) 데이터 저장 및 복구

- JSON 파일 기반 저장
- 프로그램 시작 시 자동 복원

예시:

```
{
  "target_time":"2026-06-02T15:30:00"
}
```

---

### (5) 자동 실행 등록

- Startup 폴더 자동 등록
- 바로가기(.lnk) 생성

예시:

```
C:\Users\User\AppData\Roaming\
Microsoft\Windows\
Start Menu\Programs\Startup
```
